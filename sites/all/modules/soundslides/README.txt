-- SUMMARY --

Soundslides project allows users to upload a valid soundslides project via media browser(provided by media module) in the form of a zip file and provides media formatters to render them.

For a full description of the module, visit the project page:
  http://drupal.org/project/soundslides
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/soundslides


-- REQUIREMENTS --

* Media
* Wysiwyg

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

-- CONFIGURATION --

* Setup Wysiwyg editor as per the instructions on http://drupal.org/project/wysiwyg.

* Go to admin/structure/file-types/manage/soundslides/file-display, and select soundslides preview for
  display type preview and soundslides player for other display types.
